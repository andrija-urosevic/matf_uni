# Razvoj Softvera: Ispitna Pitanja i Odgovori

1. Који од показивача `p1`, `p2` и `p3` није исправно дефинисан у 
   примеру?

```{c++}
int* p1,p2;
int* p3=(int*)1000; 
```

**Odgovor**: `p3` nije ispravno definisan.

2. У каквом су односу дужине низова s1 и s2 у примеру:

```{c++}
char s1[] = "C++";
char s2[] = { 'C', '+', '+' };
```

**Odgovor**: 1:1
